# XDU-ClassSchedule-xsl2csv

西电课程表转 csv 工具，方便导入 Google 日历等

## Requirement

- php
- composer

## How to use

```
git clone https://github.com/ShadowySpirits/XDU-ClassSchedule-xsl2csv.git
composer install
php -S locohost:80
```

[Live example](https://blog.sspirits.top/tools/xdu-class-schedule.html)
